Setup guide
1
Create a repo on github.com/new. Initialise with a README so the branch exists.
2
Generate a classic token at Settings → Tokens (classic). Tick repo scope and copy it.
3
Enter your username, repo name, and token above. Works with any GitHub account and any repo.
4
Choose ZIP mode to upload all files from a ZIP, or Custom files to pick individual files and set their destination paths in the repo.
5
Optionally set a tag like v2.0.0 (the v prefix is required by jsDelivr) and click Upload to GitHub.
6
Copy the CDN URLs that appear and paste them into your site. Use Flush CDN after re-using the same tag.
